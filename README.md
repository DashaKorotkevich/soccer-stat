# Soccer Stat

Приложение для просмотра футбольной статистики с использованием API football-data.org

## Функциональность

- Список лиг с поиском и адаптивной пагинацией
- Список команд с поиском и адаптивной пагинацией
- Матчи лиги с фильтром по дате и пагинацией
- Матчи команды с фильтром по дате и пагинацией
- Поиск по лигам и командам
- Полностью адаптивный дизайн (мобильные, планшеты, десктоп)
- Автоматическое преобразование времени из UTC в локальное

## Технологии

- React 18
- TypeScript
- Vite
- React Router v6
- Axios
- CSS Modules
